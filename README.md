Student Grade Calculator

A simple Python-based Student Grade Calculator that computes total marks, percentage, and grades for students using Object-Oriented Programming (OOP) principles and file handling.

Features

* Add and manage student records
* Calculate total marks and percentage automatically
* Generate grades based on performance
* Store student data in a text file for future reference
* View all saved records
* Search student records by roll number
* Input validation for accurate data entry

Technologies Used

* Python
* Object-Oriented Programming (Classes & Objects)
* File Handling
* Conditional Statements
* Functions

Learning Outcomes

This project helped in understanding:

* Python fundamentals
* OOP concepts
* Data validation
* File operations (read/write)
* Program modularization and code organization

Future Enhancements

* GUI using Tkinter
* Database integration with SQLite/MySQL
* Export records to Excel/CSV
* Student ranking system
* Performance analytics dashboard

How to Run

1. Clone the repository.
2. Open the project folder in a terminal.
3. Run:

python main.py

4. Follow the on-screen menu to add, view, and search student records.
